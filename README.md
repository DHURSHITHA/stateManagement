# Global Cart State Using React Context & MUI

This project demonstrates how to manage **global application state** using **React Context API** by building a simple **shopping cart** application.  
Material UI (MUI) is used to create a clean and modern user interface.

---

##  Objective

To manage shared application state (shopping cart) across multiple components using **React Context**, avoiding props drilling.

---

##  Features

- Global cart state using React Context
- Add items to cart
- Remove items from cart
- Display total cart amount
- Show **“Cart is empty”** when no items are added
- Modern UI using **Material UI (MUI)**

---

##  Technologies Used

- React.js
- React Context API
- Material UI (MUI)
- JavaScript (ES6)

---

## ⚙️ Installation & Setup

1. Clone the repository:

2. 
cd statemanagement

3. 
npm install

4.
npm install @mui/material @emotion/react @emotion/styled

5.
